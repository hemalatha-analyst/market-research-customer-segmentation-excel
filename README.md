# 📊 Market Research & Customer Segmentation Tool (Excel)

## 📌 Project Overview
This project is an Advanced Excel-based dashboard designed to analyze customer data and segment customers for better marketing and business decision-making.

It helps identify high-value customers, understand purchasing behavior, and improve targeting strategies.

---

## 🎯 Objective
- Analyze customer demographics and behavior  
- Segment customers into meaningful groups  
- Track key performance metrics (KPIs)  
- Support data-driven marketing decisions  

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers & Timeline  
- Advanced Excel Formulas  

---

## 📂 Dataset Description
The dataset includes:
- Customer ID  
- Age, Gender, Region  
- Income Level & Occupation  
- Purchase Frequency  
- Average Order Value  
- Total Spending  
- Purchase Channel  
- Engagement Level  
- Customer Type  

---

## 📊 Key Features
- Customer Segmentation (Premium, Standard, Budget)  
- KPI Metrics (Total Customers, Revenue, AOV, Repeat %)  
- Interactive Dashboard  
- Pivot Table Analysis  
- Region & Channel Performance Insights
